<h1>Installation</h1>

<li>git clone https://github.com/lauriy91/AIOHOSTING.git</li>
<li>cd AIOHOSTING</li>
<li>npm install</li>
<li>npm run build</li>
<li>npm start</li>

<h2>run front</h2>
 <li>npm run serve</li>
 
<h2>run back</h2>
<li>npm run dev</li>
